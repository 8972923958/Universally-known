# Universally Know

Universally Know is a custom trading card game featuring Monsters, Healers, Spells, Traps, and Treasure Cards.

## Win Condition

Take the final Treasure Card from the Treasure Deck.

## Features

- 50-card decks
- 9 unique species
- Monster, Healer, Spell, and Trap cards
- Shield system
- Treasure victory mechanic
- Custom card abilities# Universally-known
